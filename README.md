# 📚 BookSpark — Эхлэх цэг (Starter)

Solo Spark Vibe Coding Academy · **Курс 1**. Энэ бол **дижитал номын дэлгүүр** (BookSpark)-ийг
Claude Code-д **prompt өгч** бүтээж эхлэх цэг. Бэлэн төсөл биш — чи кодыг өөрөө prompt-оор үүсгэнэ.

## 🚀 Хэрхэн эхлэх вэ

1. **Татаж ав:**
   - Ногоон **`Code`** товч → **Download ZIP**, эсвэл
   - `git clone https://github.com/dokind/bookspark-starter.git`
2. Хавтсыг **VSCode**-оор нээ. Terminal нээгээд (`` Ctrl+` ``):
   ```bash
   bash setup.sh   # Next.js 16 хоосон апп-ыг энэ хавтсанд босгоно (~1 минут)
   claude          # Claude Code session нээнэ
   ```
3. `course/course-01-bookspark-PROMPTS.md`-ыг нээгээд prompt-уудыг **дарааллаар** өг:
   **1-р PROMPT (CLAUDE.md) → Part 1 → 2 → 3 → 4**
4. Шалгах: `npm run dev` → [http://localhost:3000](http://localhost:3000)

## 📂 Юу багтсан бэ
- `CLAUDE.md` — төслийн дүрэм (1-р PROMPT-оор сайжруулна)
- `setup.sh` — хоосон Next.js апп босгогч
- `START-HERE.md` — товч заавар
- `course/` — Курс 1-ийн хичээлүүд + Prompt Pack

## 🧱 Stack
Next.js 16 · React 19 · Tailwind CSS v4 · TypeScript
(дараагийн курсүүдэд: Supabase, QPay, Anthropic API)
